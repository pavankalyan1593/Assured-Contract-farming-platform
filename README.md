# Assured-Contract-farming-platform
Developed a web-based platform enabling farmers and buyers to connect through digital  contract farming agreements. Built key modules for user authentication, contract creation, order tracking, and secure  payment integration using Django. Ensured fair pricing and transparency by minimizing middlemen and supporting pre defined agreements.
